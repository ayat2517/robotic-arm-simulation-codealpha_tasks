PROJECT OBJECTIVES

This project simulates a 5-DOF articulated robotic arm performing an automated pick-and-place operation — a fundamental task in industrial robotics used in manufacturing, logistics, and medical device assembly.

PRIMARY OBJECTIVES

  • Design and simulate a multi-DOF robotic arm
  • Implement forward kinematics for arm control
  • Demonstrate pick-and-place motion sequencing
  • Analyze joint types and their motion constraints
  • Visualize real-time joint telemetry and arm state

TECHNICAL GOALS

  • Define workspace envelope and reachability
  • Implement smooth trajectory interpolation
  • Simulate gripper open/close actuation
  • Log operational phases with timestamps
  • Evaluate DOF requirements for task completion

DEGREES OF FREEDOM — 5-DOF ANALYSIS

The arm has 5 Degrees of Freedom plus 1 end-effector DOF (gripper), giving a total of 6 actuated axes. This configuration allows full manipulation within a 3D workspace — sufficient for most industrial pick-and-place tasks.
1
BASE ROTATION
REVOLUTE JOINT · J1
Rotates the entire arm about the vertical Z-axis. Range: ±180°. Servo-driven with encoder feedback.
2
SHOULDER PITCH
REVOLUTE JOINT · J2
Elevates the upper arm link. Range: 0°–150°. Carries highest torque load; uses DC gear motor.
3
ELBOW FLEXION
REVOLUTE JOINT · J3
Bends the forearm relative to upper arm. Range: 0°–160°. Provides reach extension.
4
WRIST PITCH
REVOLUTE JOINT · J4
Tilts the end-effector up/down. Range: ±90°. Controls object approach angle.
5
WRIST ROLL
REVOLUTE JOINT · J5
Rotates the wrist about its own axis. Range: ±180°. Allows object orientation correction.
G
GRIPPER
PRISMATIC / PARALLEL · EE
Two-finger parallel jaw gripper. Actuated by servo. Open/close stroke: 0–60mm. Force-controlled.

PICK & PLACE — MOTION SEQUENCE

01
HOME POSITION — INITIALIZATION
All joints initialize to zero/home position. Controller performs self-check and encoder calibration. IR sensor scans for object at pickup location. System enters READY state upon confirmation.
J1=0°J2=90°J3=0°GRIPPER: OPEN
02
APPROACH — TRAJECTORY TO PICK POINT
Base rotates to align with pickup zone. Shoulder and elbow joints execute coordinated motion to position end-effector above the target object. Wrist adjusts pitch to maintain vertical approach vector. Smooth trajectory interpolation (trapezoidal velocity profile) prevents mechanical shock.

03
DESCEND — LOWERING TO GRASP HEIGHT
Shoulder joint lowers the arm vertically while elbow compensates to keep end-effector aligned. Descent speed reduces to 20% as gripper approaches the object (soft-land mode). Proximity confirmed via IR sensor threshold.
FINE MOTION CONTROLSENSOR GATED
04
GRASP — GRIPPER CLOSES
Gripper servo actuates closed. Finger stroke: 0→30mm (depending on object width). Force feedback monitors grip torque to prevent object crushing. Gripper holds at 60% closure force for typical rigid objects. Confirmation: gripper encoder position logged.
GRIPPER: CLOSINGFORCE MONITORED
05
LIFT & TRANSFER — CARRY TO DROP ZONE
Arm retracts upward to clearance height (above all obstacles). Base rotates to align with drop-off zone. Shoulder and elbow joints extend to position above the drop target. Wrist adjusts for correct object orientation at destination.

Arm lowers to drop height with same soft-land profile as pickup. Gripper opens to release object. Positional accuracy target: ±2mm. Force sensors confirm object contact with surface before full release.

WORKING PRINCIPLES & KINEMATICS

FORWARD KINEMATICS

Given joint angles θ₁…θ₅, FK computes the end-effector position (x, y, z) using the Denavit-Hartenberg (D-H) convention. Each joint frame is described by 4 parameters: link length (a), link twist (α), link offset (d), and joint angle (θ).

T = T₀¹ · T₁² · T₂³ · T₃⁴ · T₄⁵
x_ee = f(θ₁, θ₂, θ₃, θ₄, θ₅)
y_ee = g(θ₁, θ₂, θ₃, θ₄, θ₅)
z_ee = h(θ₁, θ₂, θ₃, θ₄, θ₅)

INVERSE KINEMATICS

Given a target position (x, y, z), IK computes required joint angles. For pick-and-place, geometric IK is used for the 3-DOF planar subchain (J2, J3, J4), while J1 is solved independently from the target's azimuth angle.

θ₁ = atan2(y_target, x_target)
θ₃ = ±acos((r²+z²-L₁²-L₂²)/(2L₁L₂))
θ₂ = atan2(z, r) - atan2(L₂sinθ₃, L₁+L₂cosθ₃)

MOTION CONTROL STRATEGY

Each servo is controlled via PWM signals (50Hz, 1–2ms pulse width). A PID controller on each joint maintains position accuracy. Trajectory planning uses a trapezoidal velocity profile: ramp-up → constant velocity → ramp-down, ensuring smooth, jerk-free motion and protecting mechanical components from stress.

CONCLUSION

This project successfully demonstrates the design, kinematic analysis, and simulation of a 5-DOF articulated robotic arm performing a pick-and-place operation. The simulation confirms that coordinated multi-joint motion with PID control and trajectory planning achieves smooth, accurate manipulation.

KEY FINDINGS

  • 5 DOF is sufficient for full pick-and-place tasks
  • Trapezoidal velocity profiles eliminate mechanical shock
  • Geometric IK provides real-time joint solutions
  • PID control maintains ±2mm positional accuracy
  • Cycle time of ~5s is competitive with industrial arms

FUTURE IMPROVEMENTS

  • Add computer vision for dynamic object detection
  • Implement path planning for obstacle avoidance
  • Upgrade to brushless DC motors for higher speed
  • Add force/torque sensor at wrist for compliant grasp
